# Bugs 开始使用

## 安装

```bash
curl -fsSL https://bugs.neaneu.top/install.sh | sh
```

## 配置

```bash
export DEEPSEEK_API_KEY=sk-...
export OPENAI_API_KEY=sk-...
export ANTHROPIC_API_KEY=sk-ant-...
export OLLAMA_BASE_URL=http://localhost:11434
```

## 启动

```bash
bugs              # 对话模式（默认）
bugs tui          # TUI 终端界面
bugs start        # 启动守护进程
bugs stop         # 停止守护进程
```

## 界面命令

| 命令 | 作用 |
|:----|:----|
| `/exit` | 退出 |
| `/clear` | 清空对话 |
| `/scene <name>` | 切换场景 |
| `/dispatch <任务>` | 派遣子Agent 并行执行 |

## 卸载

```bash
bugs stop
rm -rf ~/.bugs
rm -f ~/.bugs/bin/bugs*
```
