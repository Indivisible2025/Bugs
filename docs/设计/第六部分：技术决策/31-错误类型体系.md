# 错误类型体系

## 错误码

```rust
pub enum ErrorCode {
    // 客户端（4xx）
    InvalidArgs = 400,
    Unauthorized = 401,
    Forbidden = 403,
    NotFound = 404,
    TooManyRequests = 429,

    // 服务端（5xx）
    InternalError = 500,
    ServiceUnavailable = 503,

    // 自定义
    TaskTimeout = 1001,
    ModelUnavailable = 1002,
    BrowserCrashed = 1003,
    MemoryLimitExceeded = 1004,
    SceneNotFound = 1005,
    PluginLoadFailed = 1006,
    ConfigInvalid = 1007,
}
```

## API 错误响应

```json
{
  "error": {
    "code": "TASK_TIMEOUT",
    "message": "任务超时：调度引擎在 30 秒内未收到子Agent #3 的结果",
    "detail": { "task_id": 42, "subagent_id": 3 }
  }
}
```

## 内部错误

```rust
pub enum BugsError {
    Task(TaskError),
    Memory(MemoryError),
    Trust(TrustError),
    Browser(BrowserError),
    Model(ModelError),
    Scene(SceneError),
    Config(ConfigError),
    Io(std::io::Error),
}

impl From<BugsError> for ErrorResponse { ... }
```

## 测试策略

| 层 | 方式 | 工具 |
|:---|:----|:----|
| 单元测试 | 每个模块独立 test | `cargo test` |
| 集成测试 | 启动真实 bugs-core，通过 API 测试 | `tests/` 目录 |
| 浏览器测试 | 需要 Chromium 安装 | `#[ignore]` 默认跳过，CI 不跑 |
| 信任引擎测试 | 固定输入→固定输出断言 | 确定性算法，易测试 |
| TUI 测试 | 快照测试（渲染输出对比） | insta crate |
| 压测 | 100K 并发任务 | `tokio::spawn` 批量 + 计时 |
