# 连接方式

## 两种模式

| 模式 | 绑定 | 认证 | 典型场景 |
|:----|:----|:----|:--------|
| **本地端口** | `127.0.0.1:8742` | 自动信任 | 本机 TUI/Web/GUI |
| **远程域名** | `0.0.0.0:8742` | Token/证书 | 手机连 PC、跨设备协作 |

## 六种组合

| 设备 | 连接方式 | 前端 |
|:----|:--------|:----|
| Linux PC | 本地端口 | TUI / Web / GUI |
| Windows PC | 本地端口 | TUI / Web / GUI |
| Linux PC | 远程域名 | Web / GUI |
| Windows PC | 远程域名 | Web / GUI |
| Android 手机 | 本地端口（Termux） | APK GUI |
| Android 手机 | 远程域名（连 PC 核心） | APK GUI |

## 配置结构

```json
{
  "network": {
    "local": {
      "bind": "127.0.0.1",
      "port": 8742,
      "auto_trust": true
    },
    "remote": {
      "enabled": false,
      "bind": "0.0.0.0",
      "port": 8742,
      "domain": "",
      "auth": {
        "mode": "token",
        "token": ""
      },
      "tls": {
        "cert": "",
        "key": ""
      }
    }
  }
}
```
