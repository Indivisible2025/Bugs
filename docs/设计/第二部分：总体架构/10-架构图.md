## 数据流

```
用户输入
    │
    ▼
主Agent (Overmind) → LLM 推理 → 理解意图 → 分裂为子任务列表
    │
    ▼
调度引擎 (虫后) → 创建子Agent → 分配任务 → 管理并发
    │
    ▼
子Agent (兵虫) → 执行任务 → 调用工具/记忆 → 汇报结果
    │
    ▼
主Agent → 聚合结果 → 生成回答 → 返回用户
```
