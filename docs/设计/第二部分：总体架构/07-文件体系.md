# 文件体系

## 全局配置目录（`~/.bugs/`）

```
~/.bugs/
├── BUG.md              ← 核心：Agent 规则和上下文（必选）
├── BUG.local.md        ← 本地覆盖，不提交 git（可选）
├── ROLE.md             ← 工作模式：编排/协调/执行（必选）
├── IDENTITY.md         ← Agent 身份标识（必选）
├── TOOLS.md            ← 工具使用说明（可选）
├── MEMORY.md           ← 记忆 + 信任机制配置（必选）
├── BROWSER.md          ← 浏览器工具配置（可选）
├── SCHEDULE.md         ← 调度参数配置（可选）
├── config.json         ← 结构化参数：模型/Provider/网络/部署（必选）
└── scenes/             ← 场景配置目录
    ├── project-a/
    │   ├── BUG.md      ← 场景覆盖（可选）
    │   ├── ROLE.md
    │   ├── MEMORY.md
    │   └── TOOLS.md
    └── scene-b/
        └── ...
```

## 文件继承规则

```
BUG.local.md（本地覆盖，最高优先级）
    ↑ 覆盖
BUG.md（场景覆盖）
    ↑ 覆盖
~/.bugs/BUG.md（全局配置，基础规则）
```

## 文件职责

| 文件 | 对标 | 说明 |
|:----|:----|:----|
| BUG.md | AGENTS.md / CLAUDE.md | 核心上下文，每次对话注入 |
| BUG.local.md | CLAUDE.local.md | 本地覆盖，开发者自己用 |
| ROLE.md | 替代 SOUL.md | 硬核，定义工作模式 |
| IDENTITY.md | OpenClaw | Agent 名称、风格、标识 |
| TOOLS.md | OpenClaw | 工具使用规则 |
| MEMORY.md | 新增 | 分级记忆 + 信任参数 |
| BROWSER.md | 新增 | 浏览器反检测/并发配置 |
| SCHEDULE.md | 新增 | 调度器参数 |

## 说明文档同步

说明文档（术语表、场景说明等）随设计文档更新而更新，保持一致性。
