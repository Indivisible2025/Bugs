# 特性清单

## P0 保底线（做不到不发布）

| # | 特性 | 含义 |
|:-:|:----|:----|
| 1 | **极端低资源占用** | **核心引擎** < 500KB（运行时、调度器、配置）。**不包括**：向量索引、加载的记忆、浏览器内核进程 |
| 2 | **极端并发性能** | 调度 < 10μs，单节点百万子Agent 并行 |
| 3 | **极端弹性** | 容错、熔断、限流、秒级冷启动 |

## P1 竞争力（拉开差距）

| # | 特性 | 含义 |
|:-:|:----|:----|
| 4 | **主Agent + 子Agent 协作** | 主Agent 分裂任务 → 调度引擎管理子Agent 池 → 兵虫并行执行，星空虫族战争模式 |
| 5 | **动态适应** | 模型/策略热插拔，不停机切换 |
| 6 | **分级记忆** | 全局/场景概要/完整记忆三层隔离，主Agent 和子Agent 共享 |
| 7 | **信任机制** | 记忆附带信用评分，多源交叉验证，-∞ 到 +∞ |
| 8 | **菌毯通信** | 主Agent ↔ 子Agent 超低延迟消息 + 统一基础设施 |
| 9 | **原生浏览器** | 内置 Chromium CEF 内核的无头浏览器，AI 搜索优化 |
